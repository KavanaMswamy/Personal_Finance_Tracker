# 💰 Personal Finance Tracker (CLI Project)

A Python-based command-line application to manage personal finances.  
This project allows users to track income and expenses with full CRUD operations and persistent JSON storage.

---

## 🚀 Features

- Add income/expense transactions
- View all transactions with IDs
- Update existing transactions
- Delete transactions
- Input validation for all fields
- Persistent storage using JSON file
- Modular code structure (main.py + services.py)

---

## 🛠️ Technologies Used

- Python 3
- JSON (file handling)
- CLI (Command Line Interface)
- Functions & modular programming

---

## 📂 Project Structure
```bash
Personal_Finance_Tracker/
```
